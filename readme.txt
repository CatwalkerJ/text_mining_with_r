This repository contains a collection R scripts 
and other resources for teaching basic text mining 
methods, including a few ways to visualization
analysis results.

The content was developed by Carl G Stahmer,
Director of Data and Digital Scholarship at UC Davis
as a resource for students who wish to develop 
skills in Digital Humanities methods. 

http://www.carlstahmer.com 
http://digitalscholarslab.org
